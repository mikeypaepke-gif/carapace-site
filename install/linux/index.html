Install CARAPACE on Linux.
Two commands. ~5 minutes.

Carapace layers on top of OpenClaw — install both, pair your phone over Tailscale, and chat from anywhere. Non-destructive on existing OpenClaw setups.

Both installers are open source — read it, fork it, send PRs.

Before you start

  • An Ubuntu or Debian VPS — anything from $5/mo works. DigitalOcean, Hetzner, Vultr, Linode, Scaleway — whatever you like.
  • SSH access — ssh user@your.vps.ip needs to work. The installer runs as your user; it'll sudo when it needs to.
  • An API key for your AI provider — xAI, OpenAI, Anthropic, or Google. Each provider has its own free-tier or pay-as-you-go entry point.
  • Carapace is a shell on top of OpenClaw. OpenClaw owns the AI runtime, your API key, and your model. Carapace adds Tailscale, the iOS pairing layer, the workspace prompts, and a few helper commands. Two installs, in order — that's the whole flow.
1

Install OpenClaw

SSH into your VPS and paste this. Handles Node, npm, and the interactive provider/key/model wizard.

curl -fsSL https://openclaw.ai/install.sh | bash

Installs Node 22 (via nvm or system package), drops the openclaw CLI, and launches the onboard wizard. Pick a provider, paste your API key, choose a model. The whole openclaw side takes about 90 seconds.

Important: when openclaw onboard asks "Hatch in Terminal? [y/N]" at the end — say NO. The terminal hatch fires a chat turn that races Carapace's setup. Skip it; you'll chat from your phone in a couple minutes.
ubuntu@vps ~ — openclaw onboard
🦞 OpenClaw setup Model/auth provider: › xAI (Grok) OpenAI (GPT) Anthropic (Claude) Google (Gemini) Enter xAI API key Key: xai-••••••••••••••••••• ✓ Default model set to xai/grok-4-1-fast-non-reasoning ✓ Gateway service installed Hatch in Terminal? [y/N]: N ✓ Setup complete.

Recommended models (fastest stable per provider): xai/grok-4-1-fast-non-reasoning, openai/gpt-5-mini, anthropic/claude-haiku-4-5, google/gemini-2.5-flash. Avoid anything with beta, preview, or 4.20 in the name — they hit safety filters or stall.

2

Layer Carapace on top

One curl, no prompts beyond Tailscale auth. Verifies openclaw, sets up Tailscale serve, drops the workspace prompts, restarts the gateway, warms it up, dumps the QR.

curl -fsSL https://carapace.info/install.sh | bash
ubuntu@vps ~ — carapace install

Heads up: piping to bash means you trust the code. Read it first if you want — it's on GitHub, MIT-licensed, line-numbered. Idempotent — safe to re-run any time you want to refresh the workspace prompts or update the helper commands.

Tailscale auth (one click)

The only place where you leave the terminal during install — happens partway through Step 2 above.

(press Ctrl+C to cancel) │ └─────────────────────────────────────────────────────┘ ✓ Tailscale connected as your-vps.tail932ab9.ts.net

No Tailscale account? Sign up free at tailscale.com/start — personal use is free up to 100 devices. Tailscale gives you a secure mesh so your phone can reach your VPS anywhere, without exposing any ports to the internet.

4

Pair your iPhone

Mock QR for preview only. Scans to carapace.info — your real pair QR embeds a one-time token.

On your iPhone — one-time setup

  1. Install Carapace from the App Store.
  2. Install Tailscale from the App Store and sign in with the same account you used during the install.
  3. In the Tailscale iOS app: enable both "Always-On VPN" and "Use Tailscale DNS Settings". Wifi often resolves .ts.net hostnames without these — but cellular silently fails to pair without them on.
  4. Open Carapace → tap Connect Server → scan the QR above.

Helper commands installed

carapace-qr Re-print the pair QR + URL — handy when you reset your phone or pair a second device.
carapace-onboard Re-run OpenClaw's provider/key/model wizard. Useful if you want to swap providers or rotate keys.
carapace-prune Wipe accumulated agent reasoning trajectories so the TUI + iOS history tab stay snappy. Runs automatically at 3:30am UTC daily — safe defaults never touch your chat history. Pass --dry-run to preview, --all for power-user clean, --nuke for a true reset.
openclaw tui Terminal chat interface, built into OpenClaw. Useful for debugging — your phone's the daily driver.

What Carapace does — and doesn't — touch

Carapace is built to layer on top of an existing OpenClaw without destroying user state. If you're already chatting with an OpenClaw agent and want to add the iOS surface, every install is non-destructive by design.

Never touched
  • · Your chat history (sessions/*.jsonl)
  • · Your API keys (auth-profiles.json)
  • · Custom SOUL.md, USER.md, OPS.md, etc.
  • · A non-default IDENTITY.md (named agents stay)
  • · Existing Tailscale serve routes
Modified surgically
  • · AGENTS.md + MEMORY.md — sentinel-bounded inserts only
  • · openclaw.json caps — bumped only if your value is below ours
  • · systemd unit drop-in — adds bonjour-disable + dynamic openclaw path resolver
  • · Crontab — adds 3am gateway restart + 3:30am trajectory prune (preserves other entries)

Want a snapshot before you run? tar czf ~/openclaw-backup.tar.gz -C ~ .openclaw — restores with tar xzf ~/openclaw-backup.tar.gz -C ~.

That's it. You're live.

Your gateway's running, your phone's paired, your key's set. Open the iOS app, start talking. Optional: add a Gemini vision key in Settings → Vision if you picked a different chat provider.
